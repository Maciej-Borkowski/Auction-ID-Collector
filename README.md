<h2>Auction ID Collector</h2><p><i><b>Disclaimer:</b> This extension is an independent tool and is not affiliated with, endorsed by, or sponsored by Allegro sp. z o.o. or the Allegro.pl platform.</i></p><br/><p><b>Auction ID Collector</b> is a lightweight and efficient Chrome extension designed to help users instantly identify and collect all product offer IDs from any active <b>allegro.pl</b> page. Whether you are conducting market research or automating data collection, this tool simplifies the extraction process with a single click.</p><br/><h3>Key Features</h3><ul><li><b>Automatic Detection:</b> Scans the DOM for all valid Allegro offer patterns.</li><li><b>Export Ready:</b> Quickly view IDs in a clean list format.</li><li><b>Lightweight:</b> Minimal impact on browser performance.</li><li><b>Real-time Results:</b> Works on search results, category pages, and merchant listings.</li></ul><br/><h3>Installation</h3><p>Since this extension is optimized for your local development or personal use, follow these steps to install it:</p><ol><li>Download or clone this repository to your local machine.</li><li>Open Chrome and navigate to <b>chrome://extensions/</b>.</li><li>Enable <b>Developer mode</b> by toggling the switch in the top right corner.</li><li>Click on the <b>Load unpacked</b> button.</li><li>Select the folder containing the extension files.</li></ol><br/><h3>How to Use</h3><p>1. Open your browser and navigate to any page on <a href='https://allegro.pl'>allegro.pl</a>.<br/>2. Click the <b>Auction ID Collector</b> icon in your Chrome toolbar.<br/>3. The extension will automatically parse the page and display a list of unique offer IDs found in search results or listings.<br/>4. Copy the list for use in your spreadsheets or internal tools.</p><br/><h3>Permissions</h3><p>This extension requires the following permissions:</p><ul><li><b>activeTab:</b> To access the content of the currently open allegro.pl page.</li><li><b>scripting:</b> To run the extraction logic within the context of the page.</li></ul><br/><h3>License</h3><p>This project is licensed under the MIT License - feel free to modify and use it for your own purposes.</p>
